Pygments로 소스코드 예쁘게 만들기

뒤로가기

블로그에는 소스코드를 예쁘게 보여주는 기능이 필요하다. 이전글인 옵시디언으로 블로그 만들기에서 설명한 것처럼 이 블로그는 티스토리나 velog 등 일반적인 블로그 플랫폼을 사용하지 않고 만들어졌다. 옵시디언에 적은 문서를 바로 포스팅하는 방식이다. 블로그를 구성하는 모든 부분을 일일이 구현해줘야하기 때문에 소스코드를 이쁘게 포맷팅하는 기능도 구현했다. 이번 포스팅에서는 그 방법을 간단히 공유해보려 한다.

기존 방법

아래와 같은 소스코드를 웹에서 이쁘게 보이게 하려면 어떻게 해야할까? 예전에 티스토리를 사용할때는 hilite.me 와 같은 Code Beautify 웹에서 코드를 복사하고 만들어진 html 코드를 티스토리 에디터에 넣었던 걸로 기억한다 (이때 따로 html 편집 모드에 넣어야한다). 이런식으로 하다가 몇년 전에 티스토리에서 코드블럭이라는게 생긴걸로 기억한다. 코드블럭을 쓰면 조금 더 간편해지긴 하지만, 커스터마이징이 좀 힘들었던걸로 기억한다.

현재 방법

어쨌든 기존의 방법은 매우 번거로운 일이다. 티스토리 같은 웹 베이스 블로그를 이용해서 포스팅을 하다보면 이런 과정 자체가 너무 귀찮고 힘이든다. 다시 돌아와서, 지금 현재 하고 있는 방식은 옵시디언 경로에 있는 markdown 문서를 Python으로 읽어서 HTML을 생성하는 방식이기 때문에 위에서 필요한 Beautify Code과정을 파이썬에 내재화할수 있다. 우선 pygments를 설치해야한다

def format(source_code, source_type):
    from pygments import highlight
    from pygments.lexers import get_lexer_by_name
    from pygments.styles import get_style_by_name
    from pygments.formatters import HtmlFormatter

    # https://pygments.org/styles/
    # 위 사이트를 참고하여 원하는 스타일을 고른다
    style = get_style_by_name("nord")
    background_color = style.background_color
    lexer = get_lexer_by_name(source_type, stripall=True)
    formatter = HtmlFormatter(
        style=style,
        linenos="table",
        cssclass="code",
        wrapcode=True,
    )

    css = formatter.get_style_defs()
    result = highlight(source_code, lexer, formatter)
    import re
    css = re.sub(re.compile("/\*.*?\*/", re.DOTALL), "", css)
    css = "".join([x.strip() for x in css.split("\n")])
    html = f"""
    <div style="
            background-color: {background_color};
            padding-top: 1.25rem;
            padding-bottom: 1.25rem;
        ">
        <style scoped>{css}</style>
        @HTMLCODE
    </div>
    """
    html = " ".join([x.strip() for x in html.strip().split("\n")])
    html = html.replace("@HTMLCODE", result)
    return html
if __name__ == "__main__":
    source_code = """
    def extract_front_matter(md_text):
        import re
        front_matter_pattern = r"^---\\n(.*?)\\n---\\n"  # Regex pattern to match front matter
        front_matter_match = re.match(front_matter_pattern, md_text, re.DOTALL)
        if front_matter_match:
            return front_matter_match.group(1)
        else:
            return None
    """

    beautified_code = format(source_code=source_code, source_type="python")
    with open("output.html", "w") as f:
        f.write(beautified_code)
